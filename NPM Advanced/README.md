# NPM Advanced

### Verificando versão

```shell
npm -v
# ou
npm --version
```

### Criando um projeto de forma rápida

```shell
npm init --yes
```

### Configurando os dados para quando usar o comando acima

```shell
npm config set init-author-name adrianoleitedasilva

npm config set init-author-email adrianoleitedasilva@outlook.com

npm config set init-license MIT
```

### Comando para verificar as minhas configurações definidas no npm init

```shell
npm get
```
